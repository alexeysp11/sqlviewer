sqlviewer: SQLite Database Viewer

About

sqlviewer is an open source project for working with SQLite databases on Windows.

Here you can get source code and information about classes.

Prerequisites

  • Windows OS;
  • SQLite.

Keyboard shortcuts

  • Ctrl + Q: go to SQL query page;
  • Ctrl + T: go to Tables page;
  • Ctrl + N: create new database;
  • Ctrl + O: open existing database;
  • Ctrl + H: open Help page;
  • Ctrl + Enter: execute SQL query (on SQL query page only).
